# TapHoaMMO

A modern, high-performance marketplace for MMO (Massive Multiplayer Online) resources, automated 24/7.

## Features
- **Categorized Resource Discovery**: Effortlessly browse through accounts, proxies, tools, and more.
- **Real-time Status**: Automatic stock updates and service notifications.
- **Premium UI**: Crafted with a focus on trust and speed.
- **Responsive Design**: Fully optimized for desktop and mobile devices.

## Tech Stack
- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React

## Development
```bash
npm install
npm run dev
```
